## 📊 Описание проекта

Проект предназначен для сбора, обработки и анализа климатических данных с целью изучения их влияния на сельскохозяйственные показатели.

### 🔍 Задачи проекта:
- ✅ Сбор погодных данных через API (Open-Meteo)
- ✅ Обработка и агрегация данных (годовые/месячные значения)
- ✅ Хранение данных в реляционной БД (SQLite → PostgreSQL)
- ✅ Корреляционный анализ Пирсона
- ✅ Визуализация результатов
- ✅ Интеграция с данными Росстата

### 🎯 Целевая аудитория:
- Исследователи в области агрономии и климатологии
- Аналитики сельскохозяйственных предприятий
- Студенты и преподаватели технических вузов

## ⚡ Функционал

| Модуль | Описание |
|--------|----------|
| 📥 `generate_weather_csv.py` | Сбор погодных данных по годам (годовые значения) |
| 📅 `generate_monthly_temp.py` | Сбор помесячных данных по температуре |
| 🗄️ `step_by_step_import.py` | Поэтапный импорт данных в SQLite |
| 🔄 `migrate_sqlite_to_postgres.py` | Миграция данных из SQLite в PostgreSQL |
| 📈 `correlation_analysis.py` | Корреляционный анализ Пирсона + визуализация |
| 📊 `pearson_analysis.py` | Расширенный анализ с группировкой по регионам |
